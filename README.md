# Library Management System

This project is a Library Management System developed using Spring Boot.

## Features
- Add Books
- View Books
- Update Books
- Delete Books
- Issue Books
- Return Books
- Student Login

## Technologies Used
- Java
- Spring Boot
- Maven
- H2 Database
- JUnit Testing
- Selenium Testing
- GitHub
- Jenkins
- Podman

## Run the Application

```bash
mvn spring-boot:run
